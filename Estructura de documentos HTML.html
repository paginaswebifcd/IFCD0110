Estructura de documentos HTML

Inicio Elementos Multimedia Accesibilidad y usabilidad Herramientas de desarrollo

Un documento HTML se compone de varias partes principales:
<!DOCTYPE html> indica que se trata de HTML5; <html> es la raíz del documento; dentro de él está <head>, donde se colocan metadatos como el título y enlaces a estilos; y <body>, donde va todo el contenido visible de la página, como encabezados, párrafos, imágenes, y más.



Aquí tenemos un ejemplo de la estructura básica que tiene que tener un documento HTML:


<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejemplo de estructura HTML</title>
  </head>
  <body>
    <h1>Hola Mundo</h1>
    <p>Este es un párrafo de ejemplo.</p>
  </body>
</html>
