Inserción de imágenes y atributos en páginas web

Para mostrar imágenes en una página web se utiliza la etiqueta <img>, que permite insertar imágenes desde archivos locales o desde internet. Es una etiqueta que no necesita cierre.

Los atributos más comunes que se usan con la etiqueta <img> son:

  • src: ruta de la imagen (ejemplo: imagenes/foto.jpg o una URL externa)
  • alt: texto alternativo si no se carga la imagen
  • width: ancho en píxeles o porcentaje
  • height: alto en píxeles o porcentaje
  • title: texto que aparece al pasar el cursor
  • loading: carga diferida con loading="lazy"
  • class: clase CSS para aplicar estilos

Ejemplo básico de inserción de una imagen:

<img src="fotos/playa.jpg" alt="Playa al atardecer" width="500" height="300">

Este código muestra una imagen llamada "playa.jpg", que se encuentra en la carpeta "fotos", con un tamaño de 500x300 píxeles.


Consejos importantes:

  • Usa siempre el atributo alt para mejorar la accesibilidad.
  • Organiza tus imágenes en carpetas claras como /imagenes.
  • Optimiza su tamaño para mejorar la velocidad de carga.
  • Utiliza loading="lazy" si la imagen está fuera de la vista inicial.
