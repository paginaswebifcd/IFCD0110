Características de los lenguajes de marcas


Inicio Elementos Multimedia Accesibilidad y usabilidad Herramientas de desarrollo

Los lenguajes de marcas son sistemas utilizados para estructurar y organizar la información dentro de un documento,
especialmente en entornos web.

Se basan en el uso de etiquetas que permiten describir el contenido, como por ejemplo "p" para párrafos o "h1" para títulos.
Estos lenguajes no ejecutan instrucciones como un lenguaje de programación, sino que definen la estructura y el significado del contenido.
Una de sus principales características es la organización jerárquica, donde los elementos se anidan unos dentro de otros para formar una estructura lógica.

Además,
permiten una separación clara entre el contenido y su presentación, ya que el diseño visual suele manejarse con hojas de estilo como CSS.
Son legibles tanto por humanos como por máquinas, lo que facilita su uso y comprensión.
También pueden ser extensibles, como en el caso de XML, donde se pueden crear etiquetas personalizadas.
En el desarrollo web, el lenguaje de marcas más común es HTML,
que trabaja en conjunto con CSS y JavaScript para construir páginas web funcionales, estructuradas y visualmente atractivas.
